## 如何使用

请确保你的Python版本是3.x，Windows环境下通过CMD或者PowerShell执行下面的命令：

```python
python main.py
```

macOS环境，如果你的辑器上同时有Python2.x和Python3.x，可通过以下命令运行：

```python
python3 main.py
```
